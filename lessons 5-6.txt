while - цикл, который повторяет свое тело, пока условие == True 

while условие:
  код

код будет выполняться ПОКА условие True. если условие будет True всегда, то цикл будет бесконечным


range(a, b, c) - функция, отдающая список с числами от a до b-1 с шагом c
range(3, 10, 2) =  [3, 5, 7, 9]


строка.split(разделитель) - функция, возвращающая список из строки, разделенной по заданному разделителю
"aaabaabaaaa".split("b") == ["aaa", "aa", "aaaa"]
--- строка разделилась по символу "b" и мы получили список из остальных символов 


строка_или_список.count(элемент) - функция, возвращающая количество заданных элементов в строке или в списке, к которому была применена функция
"aaabaabaaaa".count("b") == 2


срезы - механизм языка, позволяющий разбивать строки или списки по определенному правилу 

# каждая вторая буква
a = input()
print(a[::2])

# каждая третья буква начиная со второй
a = input()
print(a[1::3])

# каждая буква начиная с третьей и заканчивая 7
a = input()
print(a[2::8])

# является ли паиндромом
a = input()
if a == a[::-1]:
    print("ПАЛ")
